# 通用沟通技能系统

这份文档把全书转成一套可以练习的技能系统。

## 1. 沟通前：判断情境

沟通前先判断五件事：

- 目的：我想完成什么？
- 关系：我想维持什么样的关系？
- 对方：对方现在可能处于什么状态？
- 风险：这句话可能造成什么误解或防卫？
- 渠道：这个内容适合当面、电话、文字还是延后说？

常见错误：

- 只想内容目标，不想关系后果。
- 只想自己要说什么，不想对方会怎么听。
- 在情绪最高点讨论最敏感的话题。

## 2. 沟通中：先听，再说

有效倾听的四级回应：

1. 最小鼓励：嗯、我在听、你继续。
2. 事实释义：你的意思是……
3. 情绪反映：听起来你有些……
4. 需求澄清：所以你现在最希望的是……

不要过早进入：

- 分析
- 建议
- 评价
- 反驳
- 讲自己的类似经历

## 3. 表达事实：区分事实、解释、感受

沟通里最容易混乱的是把三者混在一起。

事实：

“你昨天没有回复我的消息。”

解释：

“我理解成你不想理我。”

感受：

“我有点失落，也有点不安。”

更好的组合：

“你昨天没有回复我的消息。我不确定是你太忙，还是不想聊。我有点失落，所以想确认一下。”

## 4. 处理误解：使用知觉检核

知觉检核适用于任何模糊情境。

模板：

“我注意到……我不确定是……还是……你愿意说说吗？”

示例：

- “我注意到你刚才笑了一下。我不确定你是觉得这个想法有趣，还是觉得它不靠谱。你怎么看？”
- “你这周回复少了很多。我不确定你是太忙，还是我们上次聊的事让你不舒服。”

## 5. 表达不满：用清晰信息

完整结构：

- 行为：具体发生什么
- 解释：我如何理解
- 感觉：我有什么情绪
- 结果：造成什么影响
- 意图：我希望接下来怎样

示例：

“当方案临时改变但我没有提前知道时，我会理解成我的工作没有被纳入考虑。我有点被动，也会影响我安排时间。下次如果有变化，我希望能提前同步。”

## 6. 降低防卫：六个替换

评价换描述：

- “你不负责任” -> “这次约定的时间没有完成”

支配换问题导向：

- “你必须按我说的做” -> “我们看看怎样能同时满足两个目标”

策略换坦诚：

- “你今晚忙吗？” -> “我今晚需要搬东西，想问你能不能帮忙”

中立换同理：

- “这也没办法” -> “这件事确实让人不好受”

优越换平等：

- “你根本不懂” -> “我从另一个角度看这件事”

确定换暂定：

- “这绝对行不通” -> “我担心这里可能会遇到一个问题”

## 7. 支持别人：不要急着安慰

无效安慰常见形式：

- 否定感受：“别难过”
- 看轻事情：“这没什么”
- 过早乐观：“以后会好的”
- 转移焦点：“我以前也这样”
- 直接评判：“你早该想到”

更好的支持：

- “这件事对你确实重要。”
- “你现在难过是可以理解的。”
- “我先听你说完。”
- “你希望我现在陪你想办法，还是先听你讲？”

## 8. 自我袒露：控制深度和时机

自我袒露不是越多越好。判断标准：

- 这段关系是否承受得住？
- 对方是否准备好听？
- 这个信息是否对当下关系有意义？
- 我是在建设关系，还是只是在倾倒情绪？
- 我是否期待对方给出他无法给出的回应？

## 9. 冲突处理：从立场到需求

立场是“我要什么”，需求是“为什么我要这个”。

例子：

- 立场：我不想参加这个活动。
- 需求：我需要休息，也不想在不熟的人面前消耗精力。

冲突中先找需求，再找方案。

## 10. 复盘沟通：三层检查

一次沟通结束后，可以问：

内容层：

- 事情说清楚了吗？
- 对方理解我的意思了吗？

关系层：

- 对方是否感觉被尊重？
- 我是否无意中制造了羞辱、控制或忽视？

自我层：

- 我是否表达了真实需求？
- 我是否因为防卫而攻击、逃避或操控？

# The Art of Communication Distillation

这是《沟通的艺术：看入人里，看出人外》的通用蒸馏仓库。

目标不是复述原书，也不是绑定某个具体应用场景，而是把书里的沟通理论、技巧、判断原则和练习方法，整理成一套可复用的沟通能力框架。

## 内容结构

- `docs/chapter-by-chapter-distillation.md`：按原书 11 章逐章蒸馏
- `docs/communication-skill-system.md`：把全书转成可复用沟通技能系统
- `docs/concept-index.md`：核心概念索引
- `docs/practice-cards.md`：可日常练习的沟通动作卡片
- `skill/art-of-communication/SKILL.md`：可复用的通用沟通 skill

## 使用方式

可以把这套内容用于：

- 自我沟通能力训练
- 关系沟通复盘
- 团队沟通培训
- 写作、咨询、访谈、谈判、陪伴、冲突处理等场景的通用沟通参考
